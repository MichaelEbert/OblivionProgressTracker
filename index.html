Oblivion Progress Tracker

This is a site that tracks 100% progress of The Elder Scrolls IV: Oblivion.

Tips

  • You can drag and drop an oblivion save file over the checklist to read the progress of that save file.
  • Settings can be changed on the settings page
  • If you have "enable opening links in-page" set to "other window", you have to manually drag the tab that's created out to a new window.

Nirnroute

  • Clicking next also auomatically checks off the current nirnroot
  • Clicking prev also unchecks the current nirnroot
  • You can use the ← and → keys instead of the prev/next buttons
